# Настройка фонового изображения

## Как добавить фоновое изображение

1. **Поместите ваше изображение в папку `assets/`**
   - Назовите файл `background.png`
   - Рекомендуемый размер: 1080x1920 пикселей (соотношение 9:16)
   - Формат: PNG (рекомендуется) или JPG

2. **Изображение будет автоматически использоваться**
   - Все экраны теперь используют компонент `BackgroundImage`
   - Если изображение не найдено, будет показан градиентный фон
   - Декоративные элементы (сакура, фонари, иероглиф) накладываются поверх

## Структура фонового изображения

Ваше изображение должно содержать:
- Темно-синий градиентный фон
- Розовые лепестки сакуры (опционально)
- Красные китайские фонари (опционально)
- Стилизованный иероглиф "福" (опционально)

## Настройка для разных экранов

Если вы хотите использовать разные изображения для разных экранов:

1. Обновите компонент `BackgroundImage`:
```tsx
<BackgroundImage imagePath="path/to/your/image.png">
```

2. Или передайте путь через пропсы в каждый экран

## Оптимизация изображения

- Сжимайте изображение для уменьшения размера приложения
- Используйте подходящее разрешение для мобильных экранов
- Рассмотрите использование разных разрешений (1x, 2x, 3x) для разных плотностей экранов
